# vue-3-ecomemerce

Este Proyecto se implemento von vue3, vuetify, pinia y typeScript es un ecomerce sencillo pero poderoso a donde se utilizaron nuevas tecnicas

### Product

- id
- name
- image
- price
- categoryId

### Category

- categoryID
- name
- description

### Shopping Cart

- product : PRODUCTS
- amount

## Components

En la capeta component tienes:
-Subcapetas:

- Cart:
- CartCard
- ShoppingCartItem
- SummaryCard

- SiderBar:
- CategoriesOption
- OrderOption

ItemCard
ListItems
SideBar
TopBar

## Pages

- / -> Todos los Productos
- /categoty/5 -> Productos por categoria
- /cart -> Verel carrito de compras

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
